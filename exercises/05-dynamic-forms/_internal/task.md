# 📝 Задания Уровня 5

## Как работать с заданиями

1. **Откройте `Template.tsx`** — там находятся все задания уровня
2. **Найдите нужное задание** по номеру (например, `Task5_1_Template`)
3. **Выполняйте TODO-комментарии** прямо в файле
4. **Проверяйте результат** в браузере
5. **Сравните с решением** в `Solution.tsx` после выполнения

---

## Задание 5.1: useFieldArray

### Цель
Научиться работать с динамическими массивами полей.

### Требования
Создайте форму для добавления нескольких email-адресов:
1. Изначально одно поле для email
2. Кнопка "+ Добавить" добавляет новое поле
3. Кнопка "✕" удаляет конкретное поле
4. Валидация: минимум один email, формат email

### Интерфейс

```typescript
interface EmailsForm {
  emails: { value: string }[]
}
```

---

## Задание 5.2: Условные поля

### Цель
Научитьсяconditionally отображать поля.

### Требования
Создайте форму обратной связи:
1. Выбор способа связи: email / телефон / telegram
2. В зависимости от выбора отображается соответствующее поле
3. Валидация только видимого поля

---

## Задание 5.3: Зависимые поля

### Цель
Научиться создавать зависимые поля.

### Требования
Создайте форму выбора адреса:
1. Выбор страны (Россия, USA, Германия)
2. Выбор города (список городов зависит от страны)
3. При смене страны город сбрасывается
4. Валидация обоих полей

### Данные

```typescript
const citiesByCountry = {
  ru: ['Москва', 'Санкт-Петербург', 'Казань'],
  us: ['New York', 'Los Angeles', 'Chicago'],
  de: ['Berlin', 'Munich', 'Hamburg'],
}
```

---

## Задание 5.4: Wizard (multi-step форма)

### Цель
Научиться создавать многошаговые формы.

### Требования
Создайте форму заказа в 3 шага:
1. **Шаг 1**: Имя и Email
2. **Шаг 2**: Адрес доставки
3. **Шаг 3**: Комментарий к заказу

### Функционал
- Кнопки "Назад" / "Далее" / "Отправить"
- Валидация перед переходом на следующий шаг
- Индикатор текущего шага
