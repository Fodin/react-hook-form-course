# Задание 6.2: Reset и default values

## Цель

Научиться управлять значениями формы: задавать начальные значения, программно заполнять и сбрасывать форму.

## Требования

Создайте форму профиля:

1. Поля: `username`, `email`, `role` (select)
2. Кнопка "Заполнить" — заполняет тестовыми данными
3. Кнопка "Сбросить" — возвращает к начальным значениям
4. После отправки отображайте последние отправленные данные

## Чеклист

- [ ] Форма с тремя полями: `username`, `email`, `role` (select)
- [ ] Кнопка "Заполнить" заполняет все поля тестовыми данными
- [ ] Кнопка "Сбросить" возвращает все поля к начальным значениям
- [ ] После submit отображаются отправленные данные
- [ ] isDirty корректно обновляется после заполнения и сброса

## Как проверить себя

1. Нажмите "Заполнить" — все поля должны заполниться тестовыми данными
2. Нажмите "Сбросить" — все поля должны вернуться к пустым/начальным значениям
3. Заполните форму и нажмите submit — под формой должны появиться отправленные данные
4. После сброса isDirty должен стать `false`
