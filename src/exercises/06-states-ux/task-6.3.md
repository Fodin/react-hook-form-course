# Задание 6.3: Focus management

## Цель

Научиться управлять фокусом полей при возникновении ошибок валидации.

## Требования

Создайте форму с валидацией:

1. Поля: `email`, `password`, `confirm`
2. При отправке с ошибками фокус переходит к первому полю с ошибкой
3. Отображайте сообщения об ошибках под каждым полем

## Чеклист

- [ ] Форма с тремя полями: `email`, `password`, `confirm`
- [ ] Все поля обязательны
- [ ] При submit с ошибками фокус автоматически переходит на первое невалидное поле
- [ ] Сообщения об ошибках отображаются под соответствующими полями

## Как проверить себя

1. Не заполняя полей, нажмите submit — фокус должен оказаться на поле `email`
2. Заполните `email`, нажмите submit — фокус должен перейти на `password`
3. Заполните все поля корректно и нажмите submit — форма должна успешно отправиться
