# Задание 6.5: Performance оптимизация

## Цель

Понять, как `watch` влияет на ререндеры компонентов, и научиться отслеживать количество рендеров.

## Требования

Создайте форму с отслеживанием количества рендеров:

1. Текстовое поле с подпиской через `watch`
2. Счётчик рендеров компонента формы
3. Покажите, как `watch` вызывает ререндеры при каждом изменении значения

## Подсказка

```typescript
const renderCount = useRef(0)
renderCount.current++
```

## Чеклист

- [ ] Текстовое поле, значение которого отслеживается через `watch`
- [ ] Счётчик рендеров отображается на экране
- [ ] При каждом нажатии клавиши счётчик увеличивается
- [ ] Наглядно видно влияние `watch` на количество ререндеров

## Как проверить себя

1. Откройте форму — счётчик рендеров должен показывать начальное значение
2. Введите текст в поле — счётчик должен увеличиваться с каждым символом
3. Убедитесь, что значение поля отображается в реальном времени (результат `watch`)
