# 📝 Задания Уровня 6

## Задание 6.1: Dirty / Touched states

### Цель
Научиться отслеживать состояние полей.

### Требования
Создайте форму с полями `name` и `email`:
1. Отображайте статус Dirty для каждого поля (✅/❌)
2. Отображайте статус Touched для каждого поля (✅/❌)
3. Отображайте общий статус isDirty формы

### Подсказка

```typescript
const { formState: { dirtyFields, touchedFields, isDirty } } = useForm()
```

---

## Задание 6.2: Reset и default values

### Цель
Научиться управлять значениями формы.

### Требования
Создайте форму профиля:
1. Поля: username, email, role (select)
2. Кнопка "Заполнить" — заполняет тестовыми данными
3. Кнопка "Сбросить" — возвращает к начальным значениям
4. После отправки отображайте последние отправленные данные

---

## Задание 6.3: Focus management

### Цель
Научиться управлять фокусом при ошибках.

### Требования
Создайте форму с валидацией:
1. Поля: email, password, confirm
2. При отправке с ошибками фокус переходит к первому полю с ошибкой
3. Отображайте сообщения об ошибках

---

## Задание 6.4: Accessibility (ARIA)

### Цель
Научиться создавать доступные формы.

### Требования
Создайте форму с ARIA-атрибутами:
1. `aria-invalid` для полей с ошибками
2. `aria-describedby` для связи с сообщениями об ошибках
3. `role="alert"` для сообщений об ошибках
4. `noValidate` для формы

---

## Задание 6.5: Performance оптимизация

### Цель
Научиться оптимизировать производительность.

### Требования
Создайте форму с отслеживанием количества рендеров:
1. Поле text с watch
2. Счётчик рендеров формы
3. Покажите, как watch вызывает ре-рендеры

### Подсказка

```typescript
const [renderCount, setRenderCount] = useState(0)
useEffect(() => {
  setRenderCount(c => c + 1)
}, [values])
```
