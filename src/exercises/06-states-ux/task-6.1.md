# Задание 6.1: Dirty / Touched states

## Цель

Научиться отслеживать состояния полей формы: было ли поле изменено (dirty) и было ли оно в фокусе (touched).

## Требования

Создайте форму с полями `name` и `email`:

1. Отображайте статус Dirty для каждого поля (✅/❌)
2. Отображайте статус Touched для каждого поля (✅/❌)
3. Отображайте общий статус isDirty формы

## Чеклист

- [ ] Форма с двумя полями: `name` и `email`
- [ ] Рядом с каждым полем отображается Dirty-статус
- [ ] Рядом с каждым полем отображается Touched-статус
- [ ] Общий isDirty формы отображается отдельно
- [ ] После изменения значения поля его Dirty-статус меняется на ✅
- [ ] После фокуса и выхода из поля его Touched-статус меняется на ✅

## Как проверить себя

1. Откройте форму — все статусы должны быть ❌
2. Кликните в поле `name` и уберите фокус — Touched для `name` должен стать ✅
3. Введите текст в поле `email` — Dirty для `email` и общий isDirty должны стать ✅
4. Очистите поле `email` до исходного значения — Dirty для `email` должен вернуться к ❌
