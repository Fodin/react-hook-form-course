# Задание 7.3: Submit с loading/error

## Цель

Научиться обрабатывать состояние отправки формы: загрузка, ошибки, успех.

## Требования

Создайте форму обратной связи:

1. Поля: name, email, message
2. Состояние loading во время отправки
3. Обработка ошибок сервера
4. Сообщение об успешной отправке
5. Сброс формы после успешной отправки

## Подсказка

Имитация отправки с возможной ошибкой:

```typescript
const submitFeedback = async (data: FeedbackForm) => {
  await new Promise(r => setTimeout(r, 1500))
  if (Math.random() > 0.7) throw new Error('Ошибка сервера')
  return { success: true }
}
```

## Чеклист

- [ ] Форма содержит поля name, email, message
- [ ] Во время отправки кнопка показывает состояние loading
- [ ] Поля заблокированы во время отправки
- [ ] При ошибке сервера отображается сообщение об ошибке
- [ ] При успехе отображается сообщение и форма сбрасывается

## Как проверить себя

1. Заполните все поля и отправьте форму
2. Во время отправки кнопка должна показывать loading, поля — быть заблокированы
3. Отправьте форму несколько раз — при ошибке должно появиться сообщение
4. При успешной отправке форма должна сброситься и показать сообщение об успехе
