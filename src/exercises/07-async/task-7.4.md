# Задание 7.4: Debounce для автосохранения

## Цель

Научиться реализовывать автосохранение формы с debounce.

## Требования

Создайте форму черновика:

1. Поля: subject, body
2. Автосохранение через 1 секунду после последнего изменения
3. Индикатор "Сохранено" после сохранения
4. Сохранение в localStorage

## Подсказка

Простая реализация debounce:

```typescript
const debounce = (fn: Function, ms: number) => {
  let timer: ReturnType<typeof setTimeout>
  return (...args: any[]) => {
    clearTimeout(timer)
    timer = setTimeout(() => fn(...args), ms)
  }
}
```

## Чеклист

- [ ] Форма содержит поля subject и body
- [ ] При вводе текста данные сохраняются в localStorage через 1 секунду после последнего изменения
- [ ] Отображается индикатор "Сохранено" после сохранения
- [ ] При перезагрузке страницы данные восстанавливаются из localStorage

## Как проверить себя

1. Введите текст в любое поле — через 1 секунду должен появиться индикатор "Сохранено"
2. Быстро набирайте текст — сохранение должно произойти только после паузы в 1 секунду
3. Перезагрузите страницу — данные должны восстановиться
4. Проверьте localStorage в DevTools — данные должны быть там
