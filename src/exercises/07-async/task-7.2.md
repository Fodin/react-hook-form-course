# Задание 7.2: Загрузка данных (edit mode)

## Цель

Научиться загружать данные с сервера и заполнять форму для редактирования.

## Требования

Создайте форму редактирования профиля:

1. При загрузке данные fetch-ятся с сервера (имитация)
2. Форма заполняется загруженными данными
3. Кнопка "Сохранить" активна только при изменениях
4. Отображение состояния загрузки

## Подсказка

Имитация загрузки данных профиля:

```typescript
const fetchProfile = async () => {
  await new Promise(r => setTimeout(r, 1000))
  return { name: 'Иван Иванов', email: 'ivan@example.com', bio: 'Разработчик' }
}
```

## Чеклист

- [ ] При открытии формы показывается состояние загрузки
- [ ] После загрузки поля заполнены данными с сервера
- [ ] Кнопка "Сохранить" неактивна, пока не внесены изменения
- [ ] После изменения любого поля кнопка "Сохранить" становится активной
- [ ] Форма корректно отправляется с изменёнными данными

## Как проверить себя

1. Откройте форму — должно появиться состояние загрузки (спиннер/скелетон)
2. Дождитесь загрузки — поля должны быть заполнены
3. Кнопка "Сохранить" должна быть неактивна
4. Измените значение любого поля — кнопка должна стать активной
5. Нажмите "Сохранить" — форма должна отправиться
