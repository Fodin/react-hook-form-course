## Задание 7.1: Async валидация

### Цель

Научиться выполнять асинхронную валидацию.

### Требования

Создайте форму регистрации:

1. Поле username с проверкой на сервере (имитация)
2. Поле email с валидацией формата
3. При потере фокуса на username — проверка доступности
4. Отображение статуса: "Проверка..." / "✅ Доступно" / "❌ Занято"

### Имитация API

```typescript
const checkUsername = async (username: string): Promise<boolean> => {
  await new Promise(r => setTimeout(r, 500))
  return !['admin', 'user', 'test'].includes(username.toLowerCase())
}
```

## Задание 7.2: Загрузка данных (edit mode)

### Цель

Научиться загружать данные для редактирования.

### Требования

Создайте форму редактирования профиля:

1. При загрузке данные fetch-ятся с сервера (имитация)
2. Форма заполняется загруженными данными
3. Кнопка "Сохранить" активна только при изменениях
4. Отображение состояния загрузки

## Задание 7.3: Submit с loading/error

### Цель

Научиться обрабатывать состояние отправки.

### Требования

Создайте форму обратной связи:

1. Поля: name, email, message

4. Сообщение об успешной отправке
