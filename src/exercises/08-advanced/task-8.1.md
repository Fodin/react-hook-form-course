# Задание 8.1: Интеграция с UI-библиотекой

## Цель

Научиться интегрировать React Hook Form с кастомными UI-компонентами.

## Требования

Создайте форму с кастомными компонентами:

1. TextField — обёртка над input с label и error
2. Button — кнопка с состоянием loading
3. Используйте Controller для интеграции
4. Форма содержит поля: name, email, password

## Чеклист

- [ ] Компонент TextField принимает label, error и пробрасывает props в input
- [ ] Компонент Button отображает состояние loading
- [ ] Все поля подключены через Controller
- [ ] Ошибки валидации отображаются внутри TextField
- [ ] Форма корректно отправляется

## Как проверить себя

1. Оставьте поля пустыми и отправьте — ошибки должны появиться под каждым полем
2. Заполните все поля корректно и отправьте — кнопка должна показать loading
3. Убедитесь, что компоненты TextField и Button переиспользуемые
