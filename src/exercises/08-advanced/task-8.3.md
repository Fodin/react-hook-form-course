# Задание 8.3: FormContext

## Цель

Научиться разделять форму на подкомпоненты с помощью FormContext.

## Требования

Создайте многошаговую форму с FormContext:

1. PersonalStep — поля firstName, lastName
2. ContactStep — поля email, phone
3. Общий FormProvider для всех шагов
4. Навигация между шагами

## Чеклист

- [ ] FormProvider оборачивает все шаги формы
- [ ] PersonalStep получает доступ к форме через контекст
- [ ] ContactStep получает доступ к форме через контекст
- [ ] Навигация "Назад" / "Далее" переключает шаги
- [ ] Данные сохраняются при переключении между шагами
- [ ] На последнем шаге форма отправляется со всеми данными

## Как проверить себя

1. Заполните поля на первом шаге и перейдите на второй
2. Вернитесь назад — данные первого шага должны сохраниться
3. Заполните второй шаг и отправьте форму — все данные должны быть в результате
