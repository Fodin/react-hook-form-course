# Задание 8.2: Кастомные хуки

## Цель

Научиться создавать переиспользуемые хуки для работы с формами.

## Требования

Создайте хук `useFormPersist` для сохранения формы в localStorage:

1. Хук принимает имя формы и дефолтные значения
2. Возвращает `{ stored, save, clear }`
3. Используйте хук в форме статьи (title, content)

## Подсказка

Пример API хука:

```typescript
const { stored, save, clear } = useFormPersist('article-form', {
  title: '',
  content: '',
})
```

## Чеклист

- [ ] Хук `useFormPersist` принимает ключ и дефолтные значения
- [ ] `stored` возвращает сохранённые данные или дефолтные
- [ ] `save` сохраняет данные в localStorage
- [ ] `clear` очищает сохранённые данные
- [ ] Форма статьи использует хук для восстановления данных

## Как проверить себя

1. Введите текст в поля формы статьи
2. Вызовите save (например, по кнопке или автоматически)
3. Перезагрузите страницу — данные должны восстановиться
4. Нажмите "Очистить" — данные должны удалиться из localStorage
