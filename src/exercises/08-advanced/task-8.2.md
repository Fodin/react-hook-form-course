# Задание 8.2: Кастомные хуки

## Цель

Научиться создавать переиспользуемые хуки.

## Требования

Создайте хук `useFormPersist` для сохранения формы в localStorage:

1. Хук принимает имя формы и дефолтные значения
2. Возвращает `{ stored, save, clear }`
3. Используйте хук в форме статьи (title, content)
