# 📝 Задания Уровня 8

## Как работать с заданиями

1. **Откройте `Template.tsx`** — там находятся все задания уровня
2. **Найдите нужное задание** по номеру (например, `Task8_1_Template`)
3. **Выполняйте TODO-комментарии** прямо в файле
4. **Проверяйте результат** в браузере
5. **Сравните с решением** в `Solution.tsx` после выполнения

---

## Задание 8.1: Интеграция с UI-библиотекой

### Цель

Научиться интегрировать React Hook Form с UI-компонентами.

### Требования

Создайте форму с кастомными компонентами:

1. TextField — обёртка над input с label и error
2. Button — кнопка с состоянием loading
3. Используйте Controller для интеграции

---

## Задание 8.2: Кастомные хуки

### Цель

Научиться создавать переиспользуемые хуки.

### Требования

Создайте хук `useFormPersist` для сохранения формы в localStorage:

1. Хук принимает имя формы и дефолтные значения
2. Возвращает `{ stored, save, clear }`
3. Используйте хук в форме статьи (title, content)

---

## Задание 8.3: FormContext

### Цель

Научиться разделять форму на подкомпоненты.

### Требования

Создайте многошаговую форму с FormContext:

1. PersonalStep — поля firstName, lastName
2. ContactStep — поля email, phone
3. Общий FormProvider для всех шагов
4. Навигация между шагами

---

## Задание 8.4: localStorage Persistence

### Цель

Научиться сохранять данные формы.

### Требования

Создайте форму черновика письма:

1. Поля: subject, body
2. Автосохранение при изменении
3. Восстановление при загрузке
4. Очистка после отправки

---

## Задание 8.5: Финальный проект

### Цель

Применить все изученные техники.

### Требования

Создайте форму регистрации в 3 шага:

**Шаг 1: Аккаунт**

- Email (валидация + async проверка)
- Password (минимум 8 символов)
- Confirm password

**Шаг 2: Профиль**

- First name, Last name
- Avatar (загрузка файла с предпросмотром)

**Шаг 3: Настройки**

- Newsletter (checkbox)
- Notifications (checkbox)

### Функционал

- Навигация между шагами
- Валидация перед переходом
- Сохранение данных между шагами
- Финальный просмотр данных

---
