# Задание 8.4: localStorage Persistence

## Цель

Научиться сохранять и восстанавливать данные формы в localStorage.

## Требования

Создайте форму черновика письма:

1. Поля: subject, body
2. Автосохранение при изменении
3. Восстановление при загрузке
4. Очистка после отправки

## Чеклист

- [ ] Форма содержит поля subject и body
- [ ] При изменении полей данные автоматически сохраняются в localStorage
- [ ] При загрузке страницы данные восстанавливаются из localStorage
- [ ] После успешной отправки localStorage очищается
- [ ] Поля формы сбрасываются после отправки

## Как проверить себя

1. Введите текст в поля subject и body
2. Перезагрузите страницу — данные должны восстановиться
3. Отправьте форму — данные в localStorage должны очиститься
4. Перезагрузите снова — поля должны быть пустыми
