# Задание 8.5: Финальный проект

## Цель

Применить все изученные техники в комплексной форме.

## Требования

Создайте форму регистрации в 3 шага:

**Шаг 1: Аккаунт**

- Email (валидация + async проверка)
- Password (минимум 8 символов)
- Confirm password

**Шаг 2: Профиль**

- First name, Last name
- Avatar (загрузка файла с предпросмотром)

**Шаг 3: Настройки**

- Newsletter (checkbox)
- Notifications (checkbox)

Дополнительный функционал:

- Навигация между шагами
- Валидация перед переходом на следующий шаг
- Сохранение данных между шагами
- Финальный просмотр всех данных перед отправкой

## Чеклист

- [ ] Форма разбита на 3 шага с навигацией
- [ ] Email проходит async валидацию
- [ ] Пароли совпадают
- [ ] Загрузка аватара с предпросмотром работает
- [ ] Валидация срабатывает при переходе на следующий шаг
- [ ] Данные сохраняются при переключении между шагами
- [ ] На финальном экране отображаются все введённые данные

## Как проверить себя

1. Попробуйте перейти на шаг 2 без заполнения шага 1 — должны появиться ошибки
2. Заполните шаг 1, перейдите на шаг 2, вернитесь — данные шага 1 сохранены
3. Загрузите аватар — предпросмотр должен отобразиться
4. Заполните все шаги и проверьте финальный просмотр данных
5. Отправьте форму — все данные должны быть корректны
