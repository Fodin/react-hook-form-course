# Задание 2.4: Cross-field валидация

## Цель

Научиться валидировать несколько полей вместе.

## Требования

Создайте форму смены пароля:

1. `currentPassword` — текущий пароль (обязательно)
2. `newPassword` — новый пароль (минимум 8 символов)
3. `confirmPassword` — подтверждение (должно совпадать с newPassword)
4. `email` — для уведомления (обязательно)

Добавьте валидацию:

- `confirmPassword` должен совпадать с `newPassword`
- `newPassword` не должен совпадать с `currentPassword`

## Чеклист

- [ ] Все 4 поля присутствуют
- [ ] `confirmPassword` показывает ошибку при несовпадении с `newPassword`
- [ ] `newPassword` показывает ошибку при совпадении с `currentPassword`
- [ ] Форма не отправляется при невалидных данных

## Как проверить себя

1. Введите разные значения в `newPassword` и `confirmPassword` — ошибка
2. Введите одинаковые значения в `currentPassword` и `newPassword` — ошибка
3. Заполните все корректно (разные пароли, совпадающее подтверждение) — форма отправляется
