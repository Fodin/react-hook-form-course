# Задание 2.4: Cross-field валидация

## Цель

Научиться валидировать несколько полей вместе.

## Требования

Создайте форму смены пароля:

1. `currentPassword` — текущий пароль (обязательно)
2. `newPassword` — новый пароль (минимум 8 символов)
3. `confirmPassword` — подтверждение (должно совпадать с newPassword)
4. `email` — для уведомления (обязательно)

Добавьте валидацию:

- `confirmPassword` должен совпадать с `newPassword`
- `newPassword` не должен совпадать с `currentPassword`
