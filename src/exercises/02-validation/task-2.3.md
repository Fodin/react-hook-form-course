# Задание 2.3: Custom валидация с validate

## Цель

Научиться создавать кастомные правила валидации.

## Требования

Создайте форму регистрации с паролем, который должен соответствовать правилам:

1. Минимум 8 символов
2. Хотя бы одна заглавная буква
3. Хотя бы одна цифра
4. Хотя бы один специальный символ (!@#$%^&\*)

Используйте объект `validate` с несколькими функциями для проверки каждого правила отдельно.

## Подсказка

Регулярные выражения для проверки символов:

```typescript
const hasUpperCase = /[A-Z]/
const hasDigit = /\d/
const hasSpecial = /[!@#$%^&*]/
```

## Чеклист

- [ ] Поле пароля присутствует
- [ ] Каждое правило валидируется отдельной функцией
- [ ] Ошибки показываются для каждого нарушенного правила
- [ ] Пароль `Passw0rd!` проходит все проверки

## Как проверить себя

1. Введите `short` — ошибка (менее 8 символов)
2. Введите `longenough` — ошибка (нет заглавной)
3. Введите `Longenough` — ошибка (нет цифры)
4. Введите `Longeno1` — ошибка (нет спецсимвола)
5. Введите `Passw0rd!` — все проверки пройдены
