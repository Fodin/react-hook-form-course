# Задание 1.4: formState — ошибки и валидность

## Цель

Научиться отображать состояние формы.

## Требования

1. Создайте форму с полями `email` и `password`
2. Отображайте:
   - Индикатор валидности формы (зелёный/красный)
   - Счётчик ошибок: "Ошибок: 0"
   - Статус отправки: "Отправка..." во время submit
   - Статус dirty: "Форма изменена" если поля менялись
3. Используйте `formState` для получения состояния

## Чеклист

- [ ] Поля `email` и `password` зарегистрированы с валидацией
- [ ] Индикатор валидности меняет цвет (зелёный/красный)
- [ ] Счётчик ошибок отображает актуальное количество
- [ ] Статус "Отправка..." появляется во время отправки
- [ ] Статус dirty корректно отслеживает изменения полей

## Как проверить себя

1. Откройте форму — индикатор должен быть красным (форма невалидна)
2. Введите данные в поля — статус dirty должен показать "Форма изменена"
3. Заполните поля некорректно — счётчик ошибок должен обновиться
4. Заполните все поля корректно — индикатор должен стать зелёным
5. Отправьте форму — должен кратковременно появиться статус "Отправка..."
