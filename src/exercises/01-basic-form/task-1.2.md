# Задание 1.2: Отслеживание значений с watch

## Цель

Научиться использовать `watch` для реактивного обновления UI.

## Требования

1. Создайте форму с полями `username` и `password`
2. В реальном времени отображайте:
   - Длину username
   - "Силу" пароля (слабый/средний/сильный на основе длины)
3. Используйте `watch` для отслеживания изменений

## Чеклист

- [ ] Поля `username` и `password` зарегистрированы
- [ ] Длина username обновляется при вводе
- [ ] Сила пароля меняется в зависимости от длины
- [ ] UI обновляется в реальном времени без отправки формы

## Как проверить себя

1. Введите имя пользователя — под полем должна отображаться длина
2. Введите короткий пароль (1-5 символов) — должно показать "Слабый"
3. Введите средний пароль (6-9 символов) — должно показать "Средний"
4. Введите длинный пароль (10+ символов) — должно показать "Сильный"
