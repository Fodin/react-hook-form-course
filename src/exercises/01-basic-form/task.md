# 📝 Задания Уровня 1

## Как работать с заданиями

1. **Откройте `Template.tsx`** — там находятся все задания уровня
2. **Найдите нужное задание** по номеру (например, `Task1_1_Template`)
3. **Выполняйте TODO-комментарии** прямо в файле
4. **Проверяйте результат** в браузере
5. **Сравните с решением** в `Solution.tsx` после выполнения

## Задание 1.1: Форма регистрации с разными типами полей

### Цель

Научиться регистрировать поля разных типов.

### Требования

Создайте форму регистрации с полями:

1. `firstName` — текст (обязательное)
2. `lastName` — текст (обязательное)
3. `email` — email (обязательное)
4. `age` — число (обязательное, от 18 до 100)
5. `bio` — textarea (необязательное)
6. `website` — URL (необязательное)

При отправке выводите данные в консоль и отображайте на странице.

### Интерфейс данных

```typescript
interface RegistrationForm {
  firstName: string
  lastName: string
  email: string
  age: number
  bio?: string
  website?: string
}
```

## Задание 1.2: Отслеживание значений с watch

### Цель

Научиться использовать `watch` для реактивного обновления UI.

### Требования

1. Создайте форму с полями `username` и `password`
2. В реальном времени отображайте:
   - Длину username
   - "Силу" пароля (слабый/средний/сильный на основе длины)
3. Используйте `watch` для отслеживания изменений

### Пример UI

```
Username: [john_doe]
Длина: 8 символов

Password: [******]
Сила пароля: Сильный ✓
```

## Задание 1.3: setValue и getValues

### Цель

Научиться программно управлять значениями формы.

### Требования

1. Создайте форму с полями: `title`, `description`, `price`
2. Добавьте кнопки:
   - "Заполнить тестовыми данными" — заполняет форму preset значениями
   - "Удвоить цену" — умножает значение price на 2
   - "Очистить форму" — сбрасывает все поля
3. Используйте `setValue` для изменения значений
4. Используйте `getValues` для чтения текущих значений

## Задание 1.4: formState — ошибки и валидность

### Цель

Научиться отображать состояние формы.

### Требования

1. Создайте форму с полями `email` и `password`
2. Отображайте:
   - Индикатор валидности формы (зелёный/красный)
   - Счётчик ошибок: "Ошибок: 0"
   - Статус отправки: "Отправка..." во время submit
   - Статус dirty: "Форма изменена" если поля менялись
3. Используйте `formState` для получения состояния

### Подсказка

```typescript
const {
  formState: { errors, isValid, isSubmitting, isDirty },
} = useForm()
```
