# Exercise 1.3: setValue and getValues

## Goal

Learn to programmatically manage form values.

## Requirements

1. Create a form with fields: `title`, `description`, `price`
2. Add buttons:
   - "Fill with Test Data" — fills the form with preset values
   - "Double Price" — multiplies the price value by 2
   - "Clear Form" — resets all fields
3. Use `setValue` to change values
4. Use `getValues` to read current values

## Checklist

- [ ] `title`, `description`, `price` fields are registered
- [ ] "Fill with Test Data" button fills all fields
- [ ] "Double Price" button correctly doubles the current price
- [ ] "Clear Form" button resets all fields

## How to verify

1. Click "Fill with Test Data" — all fields should be filled
2. Click "Double Price" — the price should double
3. Click "Double Price" again — the price should double again
4. Click "Clear Form" — all fields should become empty
