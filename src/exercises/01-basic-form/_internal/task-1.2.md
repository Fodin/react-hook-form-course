# Задание 1.2: Отслеживание значений с watch

## Цель
Научиться использовать `watch` для реактивного обновления UI.

## Требования
1. Создайте форму с полями `username` и `password`
2. В реальном времени отображайте:
   - Длину username
   - "Силу" пароля (слабый/средний/сильный на основе длины)
3. Используйте `watch` для отслеживания изменений

## Пример UI

```
Username: [john_doe]
Длина: 8 символов

Password: [******]
Сила пароля: Сильный ✓
```
