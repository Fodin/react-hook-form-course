# Задание 1.3: setValue и getValues

## Цель

Научиться программно управлять значениями формы.

## Требования

1. Создайте форму с полями: `title`, `description`, `price`
2. Добавьте кнопки:
   - "Заполнить тестовыми данными" — заполняет форму preset значениями
   - "Удвоить цену" — умножает значение price на 2
   - "Очистить форму" — сбрасывает все поля
3. Используйте `setValue` для изменения значений
4. Используйте `getValues` для чтения текущих значений

## Чеклист

- [ ] Поля `title`, `description`, `price` зарегистрированы
- [ ] Кнопка "Заполнить тестовыми данными" заполняет все поля
- [ ] Кнопка "Удвоить цену" корректно удваивает текущую цену
- [ ] Кнопка "Очистить форму" сбрасывает все поля

## Как проверить себя

1. Нажмите "Заполнить тестовыми данными" — все поля должны заполниться
2. Нажмите "Удвоить цену" — цена должна удвоиться
3. Нажмите "Удвоить цену" ещё раз — цена должна удвоиться снова
4. Нажмите "Очистить форму" — все поля должны стать пустыми
