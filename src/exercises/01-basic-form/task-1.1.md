# Задание 1.1: Форма регистрации с разными типами полей

## Цель

Научиться регистрировать поля разных типов.

## Требования

Создайте форму регистрации с полями:

1. `firstName` — текст (обязательное)
2. `lastName` — текст (обязательное)
3. `email` — email (обязательное)
4. `age` — число (обязательное, от 18 до 100)
5. `bio` — textarea (необязательное)
6. `website` — URL (необязательное)

При отправке выводите данные в консоль и отображайте на странице.

```typescript
interface RegistrationForm {
  firstName: string
  lastName: string
  email: string
  age: number
  bio?: string
  website?: string
}
```

## Чеклист

- [ ] Все 6 полей созданы с правильными типами (`input`, `textarea`)
- [ ] Обязательные поля отмечены через валидацию
- [ ] Поле `age` принимает только числа от 18 до 100
- [ ] Данные выводятся в консоль при отправке
- [ ] Данные отображаются на странице после отправки

## Как проверить себя

1. Заполните все обязательные поля корректными данными
2. Нажмите кнопку отправки
3. Убедитесь, что данные появились в консоли и на странице
4. Попробуйте отправить форму с пустыми обязательными полями — форма не должна отправиться
