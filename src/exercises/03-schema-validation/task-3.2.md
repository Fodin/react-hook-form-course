# Задание 3.2: Валидация с Yup

## Цель

Научиться использовать Yup схемы с React Hook Form.

## Требования

Перепишите форму из задания 3.1 используя Yup вместо Zod:

1. `email` — email, обязательно
2. `password` — минимум 8 символов, обязательно
3. `confirmPassword` — должно совпадать с password
4. `age` — число от 18 до 120

Используйте `yupResolver` для интеграции с react-hook-form.

Сравните ощущения от использования двух библиотек.

## Чеклист

- [ ] Yup схема описывает все 4 поля
- [ ] `yupResolver` передан в `useForm`
- [ ] `confirmPassword` проверяется на совпадение с `password`
- [ ] Ошибки валидации отображаются под полями
- [ ] Поведение формы идентично заданию 3.1

## Как проверить себя

1. Повторите все проверки из задания 3.1
2. Убедитесь, что ошибки и поведение совпадают
3. Обратите внимание на разницу в синтаксисе Yup vs Zod
