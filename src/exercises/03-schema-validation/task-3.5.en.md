# Exercise 3.5: Zod vs Yup Comparison

## Goal

Understand the differences between Zod and Yup and choose the right library.

## Requirements

Write a short analysis (as a code comment):

1. What did you like more about Zod?
2. What did you like more about Yup?
3. Which library would you choose for a project and why?

## Checklist

- [ ] Analysis includes syntax comparison
- [ ] Pros and cons of each library are described
- [ ] A reasoned choice is made

## How to verify

1. Re-read your analysis — does it reflect your real experience from exercises 3.1-3.4?
2. Could you explain your choice to a colleague?
