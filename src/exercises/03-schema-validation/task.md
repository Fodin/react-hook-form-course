## Задание 3.1: Базовая валидация с Zod

### Цель
Научиться использовать Zod схемы с React Hook Form.

### Требования
Создайте форму регистрации с Zod схемой:
1. `email` — email, обязательно
2. `password` — минимум 8 символов, обязательно
3. `confirmPassword` — должно совпадать с password
4. `age` — число от 18 до 120

Используйте `zodResolver` для интеграции с react-hook-form.

### Подсказка

```typescript
import { z } from 'zod'
import { zodResolver } from '@hookform/resolvers/zod'

const schema = z.object({
  // ваша схема
})

type FormData = z.infer<typeof schema>
```

## Задание 3.3: Сложные схемы

### Цель
Научиться создавать сложные схемы с объектами, массивами и enum.

### Требования
Создайте форму анкеты пользователя:

1. **Личная информация** (объект):
   - firstName: строка, обязательно
   - lastName: строка, обязательно
   - age: число, 18+

2. **Контакты** (массив объектов):
   - type: 'email' | 'phone' | 'telegram'
   - value: строка, обязательно

3. **Навыки** (массив строк, минимум 1):
   - Минимум один навык

4. **Роль** (enum):
   - 'developer' | 'designer' | 'manager'

5. **О себе** (опционально):
   - bio: строка, максимум 500 символов

## Задание 3.4: Кастомные сообщения и refine

### Цель
Научиться создавать кастомные сообщения об ошибках.

### Требования
Создайте форму смены пароля с Zod/Yup:

1. `currentPassword` — текущий пароль
2. `newPassword` — новый пароль (минимум 8 символов, заглавная, цифра)
3. `confirmPassword` — подтверждение

Используйте `.refine()` для:
- Проверки что пароли совпадают
- Проверки что новый пароль отличается от текущего
