# Задание 3.1: Базовая валидация с Zod

## Цель

Научиться использовать Zod схемы с React Hook Form.

## Требования

Создайте форму регистрации с Zod схемой:

1. `email` — email, обязательно
2. `password` — минимум 8 символов, обязательно
3. `confirmPassword` — должно совпадать с password
4. `age` — число от 18 до 120

Используйте `zodResolver` для интеграции с react-hook-form.

## Чеклист

- [ ] Zod схема описывает все 4 поля
- [ ] `zodResolver` передан в `useForm`
- [ ] `confirmPassword` проверяется на совпадение с `password`
- [ ] Ошибки валидации отображаются под полями
- [ ] Форма не отправляется с невалидными данными

## Как проверить себя

1. Отправьте пустую форму — ошибки на обязательных полях
2. Введите невалидный email — ошибка
3. Введите пароль менее 8 символов — ошибка
4. Введите разные значения в `password` и `confirmPassword` — ошибка
5. Введите `age` = 15 — ошибка
6. Заполните все корректно — данные в консоли
