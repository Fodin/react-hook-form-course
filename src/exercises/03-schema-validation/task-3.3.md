# Задание 3.3: Сложные схемы

## Цель

Научиться создавать сложные схемы с объектами, массивами и enum.

## Требования

Создайте форму анкеты пользователя:

1. **Личная информация** (объект):
   - firstName: строка, обязательно
   - lastName: строка, обязательно
   - age: число, 18+

2. **Контакты** (массив объектов):
   - type: 'email' | 'phone' | 'telegram'
   - value: строка, обязательно

3. **Навыки** (массив строк, минимум 1):
   - Минимум один навык

4. **Роль** (enum):
   - 'developer' | 'designer' | 'manager'

5. **О себе** (опционально):
   - bio: строка, максимум 500 символов

## Чеклист

- [ ] Схема содержит вложенные объекты и массивы
- [ ] Все обязательные поля валидируются
- [ ] Массив контактов поддерживает добавление/удаление
- [ ] Навыки требуют минимум одну запись
- [ ] Роль принимает только допустимые значения
- [ ] Bio ограничен 500 символами

## Как проверить себя

1. Отправьте пустую форму — ошибки на обязательных полях
2. Добавьте контакт без значения — ошибка
3. Попробуйте отправить без навыков — ошибка
4. Введите bio длиннее 500 символов — ошибка
5. Заполните все корректно — форма отправляется
