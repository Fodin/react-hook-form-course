# Задание 0.1: Первая форма

## Цель

Создать простую форму с двумя полями: email и пароль.

## Требования

1. Создайте интерфейс `LoginForm` с полями:
   - `email: string`
   - `password: string`

2. Используйте хук `useForm<LoginForm>` для управления формой

3. Зарегистрируйте два поля через `register`:
   - Поле `email` (тип `email`)
   - Поле `password` (тип `password`)

4. Добавьте кнопку отправки с текстом "Войти"

5. При отправке формы выводите данные в консоль через `console.log`

## Чеклист

- [ ] Интерфейс `LoginForm` определён
- [ ] `useForm<LoginForm>` инициализирован
- [ ] Поля зарегистрированы через `register`
- [ ] Кнопка отправки работает
- [ ] Данные выводятся в консоль

## Как проверить себя

1. Введите email: `test@example.com`, пароль: `123456`
2. Нажмите кнопку "Войти"
3. Откройте DevTools (F12) → Console
4. Убедитесь, что в консоли появился объект `{ email: "test@example.com", password: "123456" }`
