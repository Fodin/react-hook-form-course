# 📝 Задания Уровня 0

## Задание 0.1: Первая форма

### Цель

Создать простую форму с двумя полями: email и пароль.

### Требования

1. Создайте интерфейс `LoginForm` с полями:
   - `email: string`
   - `password: string`

2. Используйте хук `useForm<LoginForm>` для управления формой

3. Зарегистрируйте два поля через `register`:
   - Поле `email` (тип `email`)
   - Поле `password` (тип `password`)

4. Добавьте кнопку отправки с текстом "Войти"

5. При отправке формы выводите данные в консоль через `console.log`

### Как выполнять задание

1. Откройте файл `Template.tsx`
2. Найдите комментарии с `// TODO:` и выполните их
3. Определите интерфейс `LoginForm`
4. Инициализируйте `useForm<LoginForm>()`
5. Создайте функцию `onSubmit`
6. Создайте форму с полями email и password

### Ожидаемый результат

```tsx
// При вводе email: "test@example.com", password: "123456"
// В консоли должно появиться:
// { email: "test@example.com", password: "123456" }
```

## Задание 0.2: Вывод данных на страницу

### Цель

Добавить отображение данных формы после отправки.

### Требования

1. Создайте состояние для хранения данных формы после отправки
2. После успешной отправки отображайте данные под формой
3. Добавьте кнопку "Очистить" для сброса отображаемых данных

### Подсказка

Используйте `useState` для хранения отправленных данных:

```tsx
const [submittedData, setSubmittedData] = useState<LoginForm | null>(null)
```

---

## ✅ Проверка

1. Форма должна отображаться на странице
2. Поля должны быть интерактивными (можно вводить текст)
3. При отправке данные должны выводиться в консоль
4. После отправки данные должны отображаться под формой

### Как проверить себя

После выполнения задания:

1. Откройте `Solution.tsx` и сравните своё решение с эталонным
2. Убедитесь, что логика работы совпадает
3. Проверьте, что форма работает в браузере
