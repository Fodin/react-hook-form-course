# Задание 0.2: Вывод данных на страницу

## Цель

Добавить отображение данных формы после отправки.

## Требования

1. Создайте состояние для хранения данных формы после отправки
2. После успешной отправки отображайте данные под формой
3. Добавьте кнопку "Очистить" для сброса отображаемых данных

## Подсказка

Используйте `useState` для хранения отправленных данных:

```typescript
const [submittedData, setSubmittedData] = useState<LoginForm | null>(null)
```

## Чеклист

- [ ] Состояние для хранения отправленных данных создано
- [ ] Данные отображаются на странице после отправки
- [ ] Кнопка "Очистить" сбрасывает отображаемые данные

## Как проверить себя

1. Заполните форму и нажмите "Войти"
2. Убедитесь, что данные появились под формой
3. Нажмите "Очистить" — данные должны исчезнуть
