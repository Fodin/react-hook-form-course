# Задание 5.2: Условные поля

## Цель

Научиться условно отображать и валидировать поля в react-hook-form.

## Требования

Создайте форму обратной связи:

1. Выбор способа связи: email / телефон / telegram
2. В зависимости от выбора отображается соответствующее поле ввода
3. Валидация применяется только к видимому полю

## Чеклист

- [ ] Отображается выбор способа связи (radio или select)
- [ ] При выборе "email" появляется поле для email
- [ ] При выборе "телефон" появляется поле для телефона
- [ ] При выборе "telegram" появляется поле для username
- [ ] Валидация работает только для текущего видимого поля
- [ ] Скрытые поля не блокируют отправку формы

## Как проверить себя

1. Выберите "email" -- появляется поле email
2. Оставьте его пустым и отправьте -- ошибка валидации
3. Переключитесь на "телефон" -- поле email исчезает, появляется поле телефона
4. Заполните телефон и отправьте -- форма отправляется без ошибок
