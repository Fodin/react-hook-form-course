# Задание 5.1: useFieldArray

## Цель
Научиться работать с динамическими массивами полей.

## Требования
Создайте форму для добавления нескольких email-адресов:
1. Изначально одно поле для email
2. Кнопка "+ Добавить" добавляет новое поле
3. Кнопка "✕" удаляет конкретное поле
4. Валидация: минимум один email, формат email

## Интерфейс

```typescript
interface EmailsForm {
  emails: { value: string }[]
}
```
