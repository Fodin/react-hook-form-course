# Задание 5.1: useFieldArray

## Цель

Научиться работать с динамическими массивами полей в react-hook-form.

## Требования

Создайте форму для добавления нескольких email-адресов:

1. Изначально одно поле для email
2. Кнопка "+ Добавить" добавляет новое поле
3. Кнопка "✕" удаляет конкретное поле
4. Валидация: минимум один email, формат email

```typescript
interface EmailsForm {
  emails: { value: string }[]
}
```

## Чеклист

- [ ] При открытии формы отображается одно поле email
- [ ] Кнопка "+ Добавить" добавляет новое поле
- [ ] Кнопка "✕" удаляет поле
- [ ] Нельзя удалить последнее оставшееся поле (или форма не отправляется без полей)
- [ ] Невалидный email показывает ошибку
- [ ] Пустое поле показывает ошибку

## Как проверить себя

1. Откройте форму -- видите одно поле email
2. Нажмите "+ Добавить" несколько раз -- появляются новые поля
3. Удалите одно поле кнопкой "✕" -- поле исчезает
4. Введите невалидный email -- отображается ошибка
5. Заполните все поля корректными email и отправьте -- данные выводятся в консоль
