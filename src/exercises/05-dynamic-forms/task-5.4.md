# Задание 5.4: Wizard (multi-step форма)

## Цель
Научиться создавать многошаговые формы.

## Требования
Создайте форму заказа в 3 шага:
1. **Шаг 1**: Имя и Email
2. **Шаг 2**: Адрес доставки
3. **Шаг 3**: Комментарий к заказу

## Функционал
- Кнопки "Назад" / "Далее" / "Отправить"
- Валидация перед переходом на следующий шаг
- Индикатор текущего шага
