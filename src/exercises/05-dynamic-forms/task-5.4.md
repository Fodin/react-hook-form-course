# Задание 5.4: Wizard (multi-step форма)

## Цель

Научиться создавать многошаговые формы с пошаговой валидацией.

## Требования

Создайте форму заказа в 3 шага:

1. **Шаг 1**: Имя и Email
2. **Шаг 2**: Адрес доставки
3. **Шаг 3**: Комментарий к заказу

Функционал:
- Кнопки "Назад" / "Далее" / "Отправить"
- Валидация перед переходом на следующий шаг
- Индикатор текущего шага

## Чеклист

- [ ] Отображается первый шаг с полями "Имя" и "Email"
- [ ] Кнопка "Далее" валидирует текущий шаг перед переходом
- [ ] Кнопка "Назад" возвращает на предыдущий шаг с сохранением данных
- [ ] Индикатор показывает текущий шаг (например, "Шаг 1 из 3")
- [ ] На последнем шаге кнопка "Отправить" отправляет все данные
- [ ] Данные всех шагов сохраняются при навигации

## Как проверить себя

1. Откройте форму -- видите шаг 1 с полями "Имя" и "Email"
2. Нажмите "Далее" без заполнения -- ошибки валидации
3. Заполните и нажмите "Далее" -- переход на шаг 2
4. Нажмите "Назад" -- данные шага 1 сохранены
5. Пройдите все шаги и нажмите "Отправить" -- все данные выводятся в консоль
