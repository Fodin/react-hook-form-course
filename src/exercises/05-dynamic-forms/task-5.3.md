# Задание 5.3: Зависимые поля

## Цель

Научиться создавать зависимые поля, где значение одного поля влияет на другое.

## Требования

Создайте форму выбора адреса:

1. Выбор страны (Россия, USA, Германия)
2. Выбор города (список городов зависит от страны)
3. При смене страны город сбрасывается
4. Оба поля обязательные

```typescript
const citiesByCountry = {
  ru: ['Москва', 'Санкт-Петербург', 'Казань'],
  us: ['New York', 'Los Angeles', 'Chicago'],
  de: ['Berlin', 'Munich', 'Hamburg'],
}
```

## Чеклист

- [ ] Select для выбора страны отображается
- [ ] Select для выбора города отображается
- [ ] Список городов меняется при смене страны
- [ ] При смене страны выбранный город сбрасывается
- [ ] Оба поля проходят валидацию на обязательность

## Как проверить себя

1. Выберите "Россия" -- в списке городов появляются Москва, Санкт-Петербург, Казань
2. Выберите "Москва", затем смените страну на "USA" -- город сбрасывается, в списке New York, Los Angeles, Chicago
3. Отправьте форму без выбора города -- ошибка валидации
4. Выберите город и отправьте -- данные выводятся в консоль
