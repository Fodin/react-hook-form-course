# Задание 4.5: Дата и время

## Цель

Научиться работать с полями даты и времени в react-hook-form.

## Требования

1. `birthDate`: поле типа date (дата рождения)
2. `appointment`: поле типа datetime-local (запись на встречу)
3. Дата рождения: пользователь должен быть старше 18 лет
4. Встреча: не может быть в прошлом

```typescript
interface DateForm {
  birthDate: string
  appointment: string
}
```

## Чеклист

- [ ] Поле даты рождения отображается
- [ ] Поле даты и времени встречи отображается
- [ ] Дата рождения меньше 18 лет назад вызывает ошибку
- [ ] Дата встречи в прошлом вызывает ошибку
- [ ] Корректные значения проходят валидацию

## Как проверить себя

1. Введите дату рождения 5 лет назад -- появляется ошибка "младше 18"
2. Введите дату рождения 20 лет назад -- ошибка исчезает
3. Введите дату встречи в прошлом -- появляется ошибка
4. Введите дату встречи в будущем -- ошибка исчезает
5. Отправьте форму -- данные выводятся в консоль
