# Задание 4.4: File Upload

## Цель

Научиться загружать файлы с валидацией в react-hook-form.

## Требования

1. Тип файла: только изображения (jpeg, png, gif)
2. Размер: максимум 2MB
3. Предпросмотр изображения после выбора
4. Отображайте ошибку при неверном формате или размере

```typescript
interface AvatarForm {
  avatar: FileList
}
```

## Чеклист

- [ ] Поле загрузки файла отображается
- [ ] При выборе изображения появляется предпросмотр
- [ ] Файл неверного формата (например, .txt) вызывает ошибку
- [ ] Файл больше 2MB вызывает ошибку
- [ ] Корректное изображение проходит валидацию

## Как проверить себя

1. Выберите PNG/JPEG файл меньше 2MB -- появляется предпросмотр
2. Выберите файл .txt -- отображается ошибка формата
3. Выберите изображение больше 2MB -- отображается ошибка размера
4. Отправьте форму с корректным файлом -- данные выводятся в консоль
