# Задание 4.2: Radio и Select

## Цель

Научиться работать с radio и select полями в react-hook-form.

## Требования

1. `gender`: radio кнопки (male, female, other)
2. `country`: select (USA, Russia, Germany, France)
3. Оба поля обязательные
4. Отображайте выбранные значения после отправки

```typescript
interface ProfileForm {
  gender: 'male' | 'female' | 'other'
  country: 'USA' | 'Russia' | 'Germany' | 'France'
}
```

## Чеклист

- [ ] Три radio кнопки для выбора пола отображаются
- [ ] Select с четырьмя странами отображается
- [ ] При отправке без заполнения оба поля показывают ошибки
- [ ] После заполнения и отправки значения корректно выводятся

## Как проверить себя

1. Откройте форму -- видите radio кнопки и select
2. Нажмите "Отправить" без выбора -- оба поля показывают ошибки
3. Выберите пол и страну, отправьте -- данные выводятся в консоль
