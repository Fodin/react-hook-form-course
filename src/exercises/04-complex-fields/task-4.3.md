# Задание 4.3: Checkbox

## Цель

Научиться работать с одиночными и множественными checkbox в react-hook-form.

## Требования

1. `agree`: одиночный checkbox "Согласен с правилами" (обязательное)
2. `skills`: множественный выбор (React, Vue, Angular, Svelte)
3. Минимум один навык должен быть выбран

```typescript
interface SkillsForm {
  agree: boolean
  skills: string[]
}
```

## Чеклист

- [ ] Одиночный checkbox "Согласен с правилами" отображается
- [ ] Четыре checkbox для навыков отображаются
- [ ] Без согласия с правилами форма не отправляется
- [ ] Без выбора хотя бы одного навыка форма не отправляется
- [ ] После заполнения всех полей данные корректно отправляются

## Как проверить себя

1. Нажмите "Отправить" без выбора -- видите ошибки на обоих полях
2. Отметьте "Согласен с правилами" -- ошибка на этом поле пропадает
3. Выберите один или несколько навыков -- ошибка пропадает
4. Отправьте форму -- данные выводятся в консоль
